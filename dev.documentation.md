## Git Practice

### CI/CD

Commit to master only when want to update the main website https://app.thexdream.net  

```git
git checkout master
git merge new-feature
git push
```

### development

```git
git checkout -b "new-feature"
git add .
git commit -m "Commit message"
git push origin new-feature
```
